# Welcome to the FlexPrice API

The FlexPrice API provides powerful tools for managing users, customers, subscriptions, and more. Use this documentation to quickly integrate with our platform and unlock its full potential.

## Authentication

All API endpoints require a Bearer token for authentication. Include your token in the `Authorization` header in every request. For example:

```bash
Authorization: Bearer <your_token>
```

### Base URL
All requests should be made to the following base URL:

```plaintext
https://api.flexprice.io/
```

### Example Response Format
All responses conform to the following general format:

```json
{
    "status": "success",
    "data": { ... },
    "message": "Optional status message"
}
```

## Available Resources

FlexPrice API consists of the following major resources:

### Users
Manage user accounts, including retrieval and updates.
- **Endpoints**: `/gw/user/me`, `/gw/user/update`

### Customers
Handle customer data, such as creating and retrieving customer profiles.
- **Endpoints**: `/gw/customers/?status=published`, `/gw/customers/submit`

### Authentication
Facilitate login and logout operations to manage user sessions.
- **Endpoints**: `/gw/auth/login`, `/gw/user/logout`

Refer to each resource's respective section for detailed API specifications and examples.
