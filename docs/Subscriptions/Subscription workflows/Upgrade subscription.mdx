## Upgrade Options

### Immediate Upgrade
- **Effective Date**: Changes take effect immediately
- **Billing**: Prorated charges for the remaining period
- **Usage**: New limits and features available instantly

### Scheduled Upgrade
- **Effective Date**: Changes apply at the next billing cycle
- **Billing**: No proration, full new plan pricing from next cycle
- **Usage**: Current plan remains active until scheduled date

## Upgrade Process

### Via Dashboard
1. Navigate to the customer's subscription details
2. Click "Upgrade Subscription" button
3. Select the target plan from available options
4. Choose upgrade timing (immediate or scheduled)
5. Review proration details and confirm

### Via API
```bash
curl -X POST "https://api.flexprice.com/v1/subscriptions/{subscription_id}/upgrade" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "target_plan_id": "plan_123",
    "effective_date": "immediate",
    "reason": "Customer requested upgrade"
  }'
```

## Proration Handling

When upgrading immediately, Flexprice automatically calculates prorated amounts:

- **Credit**: Unused portion of current plan
- **Charge**: Additional cost for new plan features
- **Net Amount**: Applied to next invoice

## Upgrade Scenarios

### Feature-Based Upgrade
- Adding new features to existing plan
- Increasing usage limits
- Enabling premium features

### Plan Tier Upgrade
- Moving from Basic to Pro plan
- Upgrading from Pro to Enterprise
- Adding additional seats or licenses

## Best Practices

1. **Communicate Changes**: Notify customers before upgrading
2. **Review Usage**: Ensure new plan meets customer needs
3. **Monitor Billing**: Verify proration calculations
4. **Update Documentation**: Keep records of upgrade reasons
