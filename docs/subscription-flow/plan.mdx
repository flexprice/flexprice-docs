## Overview

A **Plan** is a sellable pricing package that bundles:
- Price
- Billing cycle
- Feature limits

> ⚠️ At least one feature must exist before creating a plan.

---

## Plan Types

- Free
- Pro
- Business / Enterprise

---

## Create a Plan

**Path:**  
`Product Catalog → Plans → Create Plan`

### Required Fields

- **Plan Name**
- **Plan Code** (immutable)
- **Currency**
- **Billing Interval** (Monthly / Yearly)
- **Price**
- **Status** (Draft / Active)

---

## Attach Features to Plan

For each feature, configure:

- **Limit Value**
- **Overage Policy**
  - Block usage
  - Allow with overage charges
- **Reset Cycle**
  - Monthly
  - Never

---

## Example: Pro Plan

| Feature | Limit |
|------|------|
| API Requests | 100,000 / month |
| Users | 10 |
| Storage | 50 GB |
