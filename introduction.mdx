## Quick Integration

```js
// Initialize Flexprice SDK
const flexprice = new Flexprice(apiKey);

// Sync product to flexprice
await flexprice.SyncFromStripe({
  product_lookup_keys: ['pro-tier', 'basic-tier'],
  customers: "all"
});

// Sync usage to stripe
await flexprice.SyncUsageToStripe({
  customer_id: 'cus_123',
  product_id: 'prod_vwe23'
});
```
